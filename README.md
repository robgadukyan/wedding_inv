# Wedding Invitation Webpage

A full single-page wedding invitation website built with React + TypeScript + Vite.

## Placeholder Images

Place your real images in `public/assets/` using these filenames:

- `hero.jpg`
- `gallery1.jpg`
- `gallery2.jpg`
- `gallery3.jpg`
- `gallery4.jpg`

If any of those files are missing, the page now falls back to local SVG placeholders automatically.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
